# Local Skills

Project-specific Codex skills for `tt.xima.work`.

- `tt-xima-content-centers` - add or update center content safely.
- `tt-xima-ui-edits` - make Astro/Tailwind UI changes in the project's existing style.
- `tt-xima-design-review` - audit design-system consistency and component reuse.

These skills are local to the repository and are meant to reduce repeated project discovery.
