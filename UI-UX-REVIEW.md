# UI/UX Review Archive

Дата исходного ревью: 2026-06-10

Этот документ оставлен как архивный UI/UX-снимок. Он может быть полезен для будущих правок, но часть наблюдений уже устарела после изменения локалей и последующих правок проекта.

## Актуальный контекст

- Активные публичные локали: `ru` и `en`.
- Русские карточки центров являются исходными и лежат в `src/data/centers_formatted`.
- Английские карточки центров лежат в `src/data/centers_i18n/en`.
- UI-переводы лежат в `src/i18n/locales/ru.ts` и `src/i18n/locales/en.ts`.
- Основной сценарий продукта по-прежнему: быстро найти нужный центр и перейти к проверяемой информации.

## Что проверять в первую очередь

1. Главная страница должна ясно вести в каталог.
2. Каталог `/centers` должен сохранять поиск, фильтры, клавиатурную навигацию и понятные empty states.
3. Карточка центра `/centers/[id]` должна иметь корректный `<h1>`, source link, metadata и локализованные действия.
4. Английские routes должны сохранять фактический смысл русских source entries.
5. Контраст, focus states и touch targets нужно проверять на mobile и desktop.

## Исторические замечания

Ниже перечислены темы из исходного ревью, которые всё ещё стоит учитывать при будущих UI-задачах:

- явный CTA на главной;
- реальное переключение grid view в каталоге, если кнопки вида остаются в интерфейсе;
- аккуратное поведение `scrollIntoView` при фильтрации;
- понятная сводка применённых фильтров;
- отсутствие лишних декоративных эффектов;
- поддержка `prefers-reduced-motion`;
- проверка meta-description и canonical behavior на страницах центров;
- локализация visible UI copy без fallback-английского в русской версии.

Перед реализацией новых UI/UX правок сверяйте этот архив с текущим кодом. Не переносите пункты механически: часть из них могла быть уже исправлена или перестать соответствовать текущему направлению продукта.
