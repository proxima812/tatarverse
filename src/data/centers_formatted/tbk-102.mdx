# Объединенное Землячество русскоговорящих в Словакии

## Контакты

- Руководитель: Измайлов Талгат Назипович

## О центре

Талгат Измайлов — предприниматель, основатель компании Izmajlov Group Slovakia (Нитра), один из основателей организации «Община», объединяющей русскоговорящих соотечественников Нитры; член Координационного совета соотечественников Словакии по направлению бизнеса. Представляет татар среди русскоязычной диаспоры Словакии.
