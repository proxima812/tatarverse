# Общественная организация «Киевское общество татар «Ильдаш» («Соотечественник»)

## Контакты

- Руководитель: Хуснутдинов Канафия Мисбахетдинович

## О центре

Общество «Ильдаш» («Соотечественник») входит во Всеукраинский татарский культурный центр «Туган тел», объединяющий 13 региональных татарских обществ Украины; Канафия Хуснутдинов возглавляет обе организации. Киевские татары — община с давней историей, о которой писала газета «День».
