# Общественная организация «Местная Татарская национально-культурная автономия Коломенского Муниципального района Московской области»

## О центре

Автономия зарегистрирована в 2010 году. Коломна — один из городов, где с 2014 года действует филиал Татарского национального центра Московской области; автономия входит в сеть местных ТНКА Подмосковья под эгидой Региональной ТНКА МО (с 1999 года).
