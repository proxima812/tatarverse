# Общественная организация «Местная татарская национально-культурная автономия Соль-Илецкого района Оренбургской области

- Телефон: [89228271869](tel:89228271869)

## О центре

Автономия действует с 2014 года. Татары компактно проживают в Соль-Илецке (~2740 человек), сёлах Линёвка (736) и Новоилецк (468). При автономии работает кружок изучения татарского языка; по её инициативе проводятся Сабантуй, турнир по боксу памяти Мусы Джалиля и конкурс «Соль-Илецкий гармонист».
