# Общественная организация «Общество татарской культуры» Кузнецкого района Пензенской области

- Телефон: [89374254379](tel:89374254379)

## О центре

Общество действует с 2003 года. Кузнецкий район — один из центров компактного проживания татар Пензенской области: в Кузнецке работают две мечети и медресе, в районе — ещё несколько мечетей. Наиль Байбиков — соавтор материалов многотомника «История татар Пензенского края».
