# Общественное объединение «Костанайский татаро-башкирский национальный культурный центр «Дуслык»

## Контакты

- Руководитель: Бектимиров Ринат Вальтаевич (член Ассамблеи народа Казахстана)

## О центре

Костанайский татаро-башкирский этнокультурный центр «Дуслык» (БИН 950440002750) в 2025 году отметил 30-летие; работает при Доме дружбы Костанайской области. Позднее центр возглавила Танзиля Хакимова. Главный праздник — Сабантуй, ставший событием для всех жителей Костаная (в пандемию проводился онлайн).
