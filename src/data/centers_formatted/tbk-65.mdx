# Местная общественная организация «Общество татарской культуры» Каменского района Пензенской области

## Контакты

- Руководитель: Девликамов Ряшит Хакимжанович

## Адрес/Локация

- Пензенская область, Каменский район, г. Каменка, ул. Центральная, д. 3

## О центре

Общество зарегистрировано 2 июня 2003 года (ИНН 5802102119). С 2003 года председателем был Ряшит Хакимжанович Девликамов; по данным реестра, позднее организацию возглавил Марат Ряшитович Девликамов. Каменский район — один из районов компактного проживания татар Пензенской области; общество работает вместе с фольклорными коллективами района над сохранением татарского культурного наследия.
